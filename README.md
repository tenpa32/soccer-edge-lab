# Soccer Edge Lab ⚽

A machine learning platform for soccer betting analytics.

## Goals

- Predict soccer match probabilities
- Calculate Expected Value (EV)
- Backtest betting strategies
- Build a live betting dashboard
- Learn Python, Machine Learning, and Data Science

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Git
- GitHub
